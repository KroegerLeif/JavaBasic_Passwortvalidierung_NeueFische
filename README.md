## Passwortvalidierung
### Voraussetzungen für ein gutes Passwort:
1. Mindestens 8 Zeichen,
2. Mindestens eine Ziffer (0–9),
3. Groß- und Kleinbuchstaben (beides muss vorkommen),
4. nicht in einer Liste häufiger/schwacher Passwörter.
